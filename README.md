COVID-19 Data Analysis 📊

🔍 Overview

This project performs exploratory data analysis (EDA) and short-term forecasting on COVID-19 data using Python.

It analyzes trends in both India and Global data, and predicts future case trends using Facebook Prophet.

---

🚀 Key Features

- Data Cleaning and Preprocessing
- India vs Global COVID-19 Trend Analysis
- Daily Confirmed, Recovered, and Death Cases Visualization
- Interactive Graphs using Plotly
- 7-Day Forecasting using Prophet Model




🛠️ Technologies Used

- Python
- Pandas
- Plotly
- Prophet
- NumPy

📁 Project Structure

- "covid_analysis.ipynb" → Main notebook (EDA + Forecasting)
- "requirements.txt" → Required libraries

---

📊 Analysis Performed

- Time-series analysis of confirmed cases
- Daily trends and growth patterns
- Country-wise and global comparison

---

🔮 Forecasting

- Used Prophet for time-series forecasting
- Predicted COVID-19 confirmed cases for the next 7 days
- Identified trend direction (increasing / decreasing)

---

📌 Key Insights

- Confirmed and recovered cases show consistent growth trends
- Multiple waves observed in daily cases
- Forecast indicates short-term trend continuation

---

▶️ How to Run

1. Open the notebook in Google Colab
2. Install required libraries:
   pip install -r requirements.txt
3. Run all cells
